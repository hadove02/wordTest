대인동기 : 사람들이 사회적 행동을 하는 이유
욕구 : 현재의 불만족스러운 상황을 바꾸기 위해 지각, 사고,의지, 행위 등을 어떤 특정방향으로 조직하는 힘 
대인신념 : 대인관계와 대인행동에 영향을 미치는 개인의 신념
비합리적 신념 : 정신적 고통이 그에 대한 개인의 생각에서 생긴다는 신념
자동적 사고 : 어떤 환경적 사건에 대해 자기도 인식하지 못하는 사이에 떠오르는 생각
인지도식 : 일반화한 믿음이나 원칙으로 구성되어 있는 인지적 구조
신체적 매력 : 외모가 낭만적 매력을 결정짓는 주요 요인
초두효과 : 첫인상을 기억하는 것은 잘하지만 이를 없애는 것은 잘하지 못하는 것
근접성 : 물리적 거리가 가까운 것
친숙성 : 상대방과 접촉하는 빈도
단순접촉 효과 : 자주 보면 호감을 느끼게 되는 현상
과노출 효과 : 접촉 빈도가 정도 이상을 넘어서면 호감도가 감소
유사성 : 삶에 대한 태도, 가치관, 의식 구조 등이 비슷한 사람
상보성 : 자신이 지니지 못하고 있는 속성을 지니고 있는 상대에게 호감
호혜성 : 나를 좋아해 주는 사람을 나도 좋아하게 되는 것
자기충족적 예언 : 내가 상대방에게 지속적으로 어떤 기대를 하고 대하면 상대방도 그 기대에 보답하는 방식으로 반응
튕기기 전략 : 내게 호감을 보여도 여기에 쉽게 넘어가지 않을 사람
도파민 : 다른 사람과 협력하는 행동을 하면 나오는 호르몬
에로스 : 뜨거운 열정과 욕망이 중요한 요소가 되는 강렬한 사랑
스토르게 : 깊은 우정이나 애착이 주된 요소가 되는 사랑
옥시토신 : 오랜 관계에서 느껴지는 사랑의 감정, 사회적 보살핌과 이타적 행동을 유발하는 호르몬
루두스 : 재미와 쾌락을 중요시하며 즐기는 사랑
프라그마 : 여러 조건을 고려하여 사랑의 대상을 선택하는 현실주의적이고 합리주의적인 사랑
아가페 : 상대방을 위하고 보살피는 마음으로 하는 무조건적이고 헌신적인 사랑
마니아 : 강렬한 소유욕과 집착을 중요한 요소로 하는 강렬한 사랑
친밀감 : 사랑의 정서적 요인, 사랑의 따뜻한 측면
열정 : 사랑의 동기적 요인, 사랑의 뜨거운 측면
헌신 : 사랑의 인지적 요인, 사랑의 차가운 측면
매력 : 상대와 가까이하고 싶게 만드는 힘, ABCDE 단계에서 A
형성 : 서로에 대한 호감을 깊어지게 만든다, ABCDE 단계에서 B
지속 : 관계가 지속되기 위한 노력, ABCDE 단계에서 C
악화 : ABCDE 단계에서 D
종결 : 마지막 단계, ABCDE 단계에서 E
결혼전기 : 부모님과 나의 관계가 분리되는 것을 받아들이는 시기
약혼식 : 양가의 직계 가족이 모여 공식적으로 첫인사를 나누는 자리
상견례 : 양가의 어른들이 정식으로 만나 서로 인사하고 자식들의 혼사를 이야기하는 자리 
결혼적응기 : 결혼을 해서 부부 두 사람으로 가정이 꾸려지는 시기
자녀아동기 : 부부 두 사람으로 구성되었던 가정에 자녀가 태어남으로 새로운 가족 구성원이 수용 되는 시기
자녀청소년기 : 자녀가 서서히 독립(중 고등학교 재학)을 향해 나아가는 시기
자녀독립기 : 자녀가 결혼을 해서 새로운 구성원이 들어오 시기
노년기 : 역할 변화를 수용해야 하는 단계
영아기 : 영아가 주양육자와 맺는 정서적 유대인 애착관계를 형성
전애착 단계 : 사회적 신호를 통해 주변인으로 하여금 자신을 돌보게끔 하는 단계
애착형성 단계 : 자신을 정기적으로 돌보아 주는 성인에게만 사회적 교류를 기대, 유지하는 단계
제한된 애착형성 단계 : 소수의 특정한 사람들에게만 제한된 애착을 형성하는 단계
상호적 관계형성 단계 : 인지능력과 언어능력의 발달, 신체적 발달 (걸음마) 단계
안정애착 : 주양육자의 존재가 안정감을 주고, 신뢰감을 주는 애착관계
회피애착형 : 주양육자에 대해 무관심하거나 모른 척하며 아무런 반응을 보이지 않는 애착관계
저항애착형 : 주양육자가 존재하여도 불안해하고, 주양육자가 사라지면 심한 분리불안을 보이는 애착관계
혼란애착형 : 회피애착과 저항애착이 결합된 형태, 불안정애착의 가장 심한 형태
