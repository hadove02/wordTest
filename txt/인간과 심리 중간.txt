대인동기 : 개인이 타인을 향해 특정 방향으로 행동하게 하는 것
대인신념 : 대인관계와 대인행동에 영향을 미치는 개인의 신념
대인감정 : 타인과의 관계 속에서 경험하는 다양한 정서적 체험
대인사고 : 타인과의 관계 속에서 경험한 내용에 대해 자신의 평가
욕구 : 현재의 불만족스러운 상황을 바꾸기 위해 특정 방향으로 조작
초두효과 : 첫인상을 잘 기억함
근접성 : 물리적 거리가 가까운것
친숙성 : 상대방과 접촉하는 빈도
과노출 효과 : 접촉 빈도가 정도 이상을 넘어가면 호감이 감소
유사성  : 삶에 대한 태도, 가치관, 의식 구조 등이 비슷한 사람
상보성 : 자신이 지니지 못하고 있는 속성을 지니고 있는 보완적인 상대
호혜성 : 나를 좋아해 주는 사람을 나도 좋아하게 되는 것
자기충족적 예언 : 상대방에게 지속적으로 기대를 하고 대하면 상대방도 기대에 보답
가족적 동반자 : 혈연으로 맺어지는 동반자
낭만적 동반자 : 애인, 배우자와 같은 동반자
사교적 동반자 : 서로에 대한 호감 혹은 공통적인 관심사, 신뢰로 맺어진 동반자
작업적 동반자 : 특정목표, 이해관계에 기반하여 형성된 일시적인 동반자
추동 : 마음을 움직여 무엇인가 행동하게 하는 에너지
무의식적 동기 : 인간 행동의 원인이 무의식적인 요인에 의해 더 많은 영향을 받음
불안 : 개인에게 부정적인 일이 발생할 것 같을 때 나타나는 위험경보신호
현실적 불안 : 현실상황을 기준으로 발생하는 불안
신경증적 불안 : 원초아를 통제하지 못할 것 같을 때 발생하는 불안
도덕적 불안 : 도덕적 · 사회적 관점을 기준으로 발생하는 불안
방어기제 : 자아가 불안을 처리하기 위해 사용하는 기제
의식 수준 : 개인이 항상 자각하고 있는 수준
전의식 수준 : 약간의 노력을 기울이면 쉽게 의식으로 올릴 수 있는 수준
무의식 수준 : 쉽게 의식되지 않는 수준
원초아 : 쾌락원리에 의해 움직임
자아 : 현실원리에 의해 움직임
초자아 : 도덕원리에 의해 움직임
통찰 : 내면에 대한 분석을 통해 무의식을 의식화하여 이해하는 것
비합리적 신념 : 정신적 고통이 그에 대한 개인의 생각에서 생긴다는 신념
인지적 구조 : 과거 경험을 통해 자신과 세상에 대한 일반화
인지도식 : 일반화한 믿음이나 원칙으로 구성되어 있는 인지적 구조
핵심신념 : 자신과 세상 전반에 대해 과잉일반화한 경직된 내용의 신념
중간신념 : 삶에 대한 태도, 규범, 가정의 신념
자동적 사고 : 표층적인 수준의 인지
경직성 : 해당 생각이 너무 경직된 것은 아닌지
논리성 : 해당 생각이 너무 비논리적인 것은 아닌지
현실성 : 해당 생각이 너무 비현실적인 것은 아닌지
실용성 : 해당 생각이 자신에게 도움이 되는 생각인지
감각운동기 : 대상 영속성, 자신의 신체를 중심으로 사고하는 인지 발달 단계
전조작기 : 자아중심성, 표상적인 놀이의 인지 발달 단계
구체적 조작기 : 보존 개념 획득, 자아 중심성 극복, 추상적 개념 이해하는 인지 발달 단계
형식적 조작기 : 가설적 사고, 과학적 사고, 추상적 사고에 해당하는 인지 발달 단계
동화 : 새로운 정보를 기준의 인지구조에 근거해 받아들이는 인지발달 과정
조절 : 새로운 정보가 기존의 인지구조로 받아들일 수 없게되면 인지구조를 변형하는 인지발달 과정
평형 : 여러 번의 시행착오를 거쳐 새로운 도식을 만드는 인지발달 과정
조직 : 새로운 정보가 인지구조와 결합되어 인지구조가 변화하는 인지발달 과정
대상영속성 : 눈 앞의 대상이 사라져도 존재한다는 것을 아는 것 
실제적 발달 영역 : 아동이 과제를 혼자 해결할 수 있는 수준
근접 발달 영역 : 실제적 발달 수준과 잠재적 발달 수준 사이의 수준
잠재적 발달 영역 : 유능한 타인의 도움을 받아 해결할 수 있는 수준
리비도 : 삶의 본능이 만들어내는 추동
공격추동 : 죽음의 본능이 만들어내는 추동
