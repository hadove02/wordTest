컴퓨터 그래픽스 : 입력없이 영상을 만들어 출력되는 것
영상 처리 : 영상을 입력하면 잡음 제거, 화질 향상등을 거쳐 영상이 출력되는 것
컴퓨터 비전 : 영상을 입력하면 분석을 통해 묘사나 해석이 출력되는 것
해상도 : Width x Height
종횡비 : 가로-세로 비율
프레임 버퍼 : 레스터 그래픽 장치에서 화면에 출력할 내용을 보관하는 것
엘리어싱 : 이미지 등을 확대할 때 레스터 장치에서 발생하는 계단 효과
모델링 : 물체 및 장면을 정의하는 작업
렌더링 : 물체 및 장면을 그려내는 작업
광자검출기 : 빛을 전하로 변환시켜 세기를 측정하는 것
쉐도우 마스크 : 화소의 점을 정밀하게 표시하게 하기 위한 방법
인터레이싱 방식 : 홀수 필드와 짝수 필드를 번갈아가면서 전자총 쏨
프로그래시브 방식 : 전자총을 쏴서 그림
색 범위 : 특정 장치로 표현할 수 있는 색의 영역
색 온도 : 빛의 색을 절대온도를 이용하여 숫자로 표시
색상 : 빨강, 파랑, 노랑 등과 같은 색의 종류
채도 : 짙은 빨강, 옅은 빨강 등과 같은 색의 진한 정도
명도 : 밝은 빨강, 어두운 빨강 등과 같은 색의 밝기를 의미
라이브러리 : 개발자가 호출하여 사용하는 것
프레임워크 : 개발자가 작성한 코드를 사용하는 것
고수준 그래픽스 API : 장면 묘사를 위해 장면 그래프 활용하는 API
저수준 그래픽스 API : 구체적인 함수를 호출하며 하드웨어와 직접적으로 연관 됨
디스플레이 리스트 : 반복적으로 실행할 그리는 요소들을 정의하여 컴파일된 형태로 재사용
벡터 공간 : 벡터 위치와 무관하게 크기와 방향만 같으면 동일 하다고 여기는 공간
어파인 공간 : 벡터 크기 방향 위치를 전부 고려하기 위해 점을 추가한 공간
동차 좌표계 : 벡터와 점을 구분하기 위해 한 차원을 추가한 좌표계
전역 좌표계 : 모든 물체를 일률적으로 포함할 수 있는 기준 좌표계
더블 버퍼링 : 두 개의 프레임 버퍼를 사용하여 화면을 보이는 기술
Computer Generated Imagery : CGI
Charge-Coupled Device : CCD
Cathode Ray Tube : CRT
Liquid Crystal Display : LCD
Light Emitting Diode : LED
Organic Light Emitting Diode : OLED
Head Mounted Display : HMD
Computer Aided Design : CAD
Virtual Reality : 현실적인 요소가 거의 없는 확장 현실
Augmented Reality : 현실위에 가상의 콘텐츠를 덮음
Mixed Reality : 현실 세계와 가상 세계를 결합, 현실 세계와 상호작용하는 가상세계
eXtended Reality : VR, AR, MR을 모두 포괄한 개념
Hue Saturation Lightness : HSL
Hue Saturation Value : HSV
Hue Saturation Intensity : HSI
Application Programming Interface : API
Virtual Reality Modeling Language : VRML
Dot Per Inch : DPI
