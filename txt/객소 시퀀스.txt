제품 개발 프로세스 : 분석 → 설계 → 구현 → 시험
프로젝트 관리 프로세스: 범위관리 → 일정관리 → 품질관리
프로젝트 관리 주기(PDCA) : 계획 → 실행 → 확인 → 조치
PMBOK 프로세스 그룹 : 시작 → 계획수립 → 실행 → 감시/통제 → 종료
UML 사물 : 구조 → 행동 → 그룹 → 주석
UML 관계 : 의존 → 연관 → 일반화 → 실체화
UML 다이어그램 : 클래스 → 객체 → 유스케이스 → 시퀀스 → 상태 → 액티비티 → 컴포넌트 → 배치 → 커뮤니케이션
객체지향 분석 프로세스 : 행위자 → 유스케이스 다이어그램 → 유스케이스 시나리오 → 유스케이스 시나리오 정보 → 클래스 다이어그램 → 시퀀스 다이어그램 → 액티비티 다이어그램 → 요구사항 명세서
프로젝트 제안요청 : 요구 사항 분석 → 기본 설계 → 상세 설계 → 코딩 → 단위시험 → 통합시험 → 시스템시험 → 인수시험
유지보수 절차 (형상관리 o) : 설계서 분석 → 보수계획 → 설계 변경 → 코드 수정 → 테스트
유지보수 절차 (형상관리 x) : 원시코드 분석 → 변경효과 분석 → 코드 수정 → 테스트
